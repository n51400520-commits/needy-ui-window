# Needy UI Window

Проект: окно в стиле Needy Girl Overdose  
Собрано на HTML/CSS/JS для визуального прототипа стрим-интерфейса

## Функции
- Чат-блок с сообщениями
- Индикаторы настроения, энергии, подписчиков
- Кнопки управления стримом

## Запуск
Открыть `index.html` в браузере  
Или перейти по GitHub Pages: https://username.github.io/needy-ui-window
